Quick Start Guide

Get up and running with the Elevate Courses site and notes workflow.

Listening to Episodes

Option 1
In-browser

Visit Elevate Courses and hit play on any episode directly.

Option 2
Podcast app

Subscribe with any podcast app using the RSS feed URL:

https://dragonbrosy.github.io/Elevate-Courses/feed.xml

Works with Apple Podcasts, Spotify, Overcast, Pocket Casts, and any RSS-compatible app.

Reading Notes

Step 1
Browse the notes index

All ground school notes are at Elevate Courses / Notes.

Step 2
Reference pages

See Syntax Reference for Markdown and LaTeX math syntax, and Resources for FAA references and study links.

Publishing Your Own Notes

Step 1
Add frontmatter to a note

In Obsidian, add this to the top of any .md file:

---
publish: true
title: "Day 7 — Navigation"
date: 2026-03-24
topics: "VOR, GPS, pilotage"
---
Step 2
Push

Run push.sh or use the Obsidian Git plugin. GitHub Actions handles the rest automatically.

Step 3
Start sessions with a pull

Run start.sh before writing to sync any changes made on GitHub.
